Import TJBot library

from tjbot import *

Instantiate a TJBot object

bot = TJBot("192.168.1.153")

Set arm angle

TJBot.arm(self, angle)
Sets TJBot's arm servo motor to the specified integer parameter "angle".
Example:
for i in range(5): bot.arm(90) time.sleep(1) bot.arm(180) time.sleep(1)

Set LED color

led(color)
Color parameters must be string english words like "red", "green" or "yellow". The "black" argument causes the LED to turn off.
