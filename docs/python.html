Import TJBot library

from tjbot import *

Instantiate a TJBot object

bot = TJBot("IP ADDRESS")

Set arm angle

arm(angle)
Sets TJBot's arm servo motor at the specified integer parameter "angle".
Example: 
        
for i in range(5):
    bot.arm(90)
    time.sleep(1)
    bot.arm(180)
    time.sleep(1)

Set LED color

led(color)
Color parameters must be string english words like "red", "green" or "yellow" (use "black" to turn off the LED).
